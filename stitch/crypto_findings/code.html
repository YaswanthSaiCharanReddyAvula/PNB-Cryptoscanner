RSA-2048 Key Exchange

Shor's Vulnerable High Confidence

Vulnerability Description

RSA cryptography relies on the integer factorization problem. It is mathematically vulnerable to Shor's algorithm when executed on a sufficiently capable Cryptographically Relevant Quantum Computer (CRQC). Current 2048-bit keys do not provide adequate post-quantum resistance.
